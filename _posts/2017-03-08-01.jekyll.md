## 03. jekyll 폴더이동(Gemfile에 define된 dependency 설치)

```
Gemfile 예시
source 'https://rubygems.org'

gem 'jekyll', '3.3.1'

group :jekyll_plugins do
	gem 'jekyll-feed', '0.6.0'
	gem 'jekyll-seo-tag', '2.1.0'
	gem 'jekyll-paginate', '1.1.0'
	gem 'jekyll-sitemap', '1.0.0'
end
```

```
gem bundle install
```
